# deepresearch_summerschool_2026
Датасеты для летней школы.


# Описание файлов

## Задача 1

### Описание данных

Файл _summerschool_task1.csv_ . Разделитель - табуляция.

Каждый элемент колонки **question** - DeepResearch-вопрос, ответ на который искался системой.

Каждый элемент колонки **queries** - запрос к поисковой системе.

Каждый элемент колонки **links** - список ссылок, выданных системой для каждого элемента поля queries.

### Описание заданий

Для разных поисковиков - попробовать для каждого из queries получить список ссылок, оценить процент совпадения и сделать выводы. Задача разминочная, не оценивается.

## Задача 2 

### Описание данных

Файл _summerschool_task2.csv_ . Разделитель - табуляция.

Каждый элемент колонки **url** - это url-адрес сайта, парсинг которого шел.

Каждый элемент колонки **domain** - домен, полученный библиотекой tldextract для данного url-адреса. Отдельно уточняем,что качество работы системы в этой задаче оценивается взвешенно по домену.

Участники для каждого url предоставляют контент, который они спарсили. Контент предобрабатывать не нужно. Финальная оценка будет осуществляться на тестовом файле, который хранится у организаторов. Для обеспечения эффективности оценки настоятельно рекомендуется использование зарубежного / динамического IP.
